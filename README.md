# End-to-End Sales Forecasting & Demand Intelligence System

## Overview

This project predicts future sales demand using multiple forecasting techniques and provides an interactive Streamlit dashboard.

## Features

- Sales Forecasting using SARIMA, Prophet and XGBoost
- Time Series Analysis
- Anomaly Detection
- Product Demand Segmentation
- Interactive Streamlit Dashboard

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Statsmodels
- Prophet
- XGBoost
- Streamlit
- Matplotlib
- Seaborn

## Dashboard

- Sales Overview
- Forecast Explorer
- Anomaly Report
- Product Demand Segments
